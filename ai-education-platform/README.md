# AI个性化教育平台

> 基于大模型的个性化资源生成与学习多智能体系统 | 第15届中国软件杯 A3命题

## 📖 项目概述

本系统是一个面向高校的智能教育平台，通过多智能体协作，为学生提供个性化学习资源推荐、学情分析、智能作业生成等功能，同时为教师、辅导员和管理员提供全方位的教学管理支持。

## ✨ 功能特性

### 👨‍🎓 学生端
- 📚 课程管理：查看课程表、课件、重难点资源
- 📝 作业系统：在线提交作业、查看批改结果与AI反馈
- 📊 考试分析：成绩查看、薄弱点分析、AI推荐复习资源
- 🤖 个性化学习：AI根据学情推荐练习题、学习计划、课程回放
- 💬 聊天系统：同学间私聊、课程群聊
- 🏠 生活服务：请假申请、离校审批、宿舍水电费
- 🔔 智能提醒：作业截止、课程PPT、学业预警

### 👨‍🏫 教师端
- 📖 课程管理：上传课件、管理课程内容
- ✍️ 签到系统：8秒实时动态码 + 位置权限双重验证
- 📋 作业管理：发布作业、AI智能推荐题目、在线批改
- 📊 班级分析：平时分统计、学分计算、班级整体画像
- 🎯 考试分析：班级失分分析、教学重点优化建议
- 🤖 AI教学助手：根据班级学情生成教学建议

### 👩‍💼 辅导员端
- ✅ 请假审批：请假/离校外出申请审核
- 📊 学分总览：班级学生学分统计与可视化
- ⚠️ 学业预警：自动识别学业困难学生
- 📢 班级公告：发布班级通知

### 🔧 管理员端
- 🖥️ 服务器监控：CPU、内存、磁盘使用率
- 🐛 Bug管理：系统异常与Bug报告追踪
- 👥 用户管理：账号管理、角色分配

## 🤖 多智能体系统

本系统的核心创新在于**四个协作智能体**：

| 智能体 | 职责 | 输入 | 输出 |
|--------|------|------|------|
| 学情分析Agent | 分析学生学习状态 | 成绩、出勤、学习记录 | 薄弱点、学习风格、风险等级 |
| 资源生成Agent | 生成个性化学习资源 | 学生画像、薄弱知识点 | 练习题、解析、学习计划 |
| 教学优化Agent | 优化教学策略 | 班级成绩、考试数据 | 教学建议、进度调整、重点推荐 |
| 考试分析Agent | 深度分析考试情况 | 考试结果、题目数据 | 难度分析、题目质量评估、改进策略 |

**Agent协作流程：**
```
学生完成考试/作业
       │
       ▼
  考试分析Agent ──────┐
       │              │
       ▼              ▼
  学情分析Agent    教学优化Agent
       │              │
       ▼              │
  资源生成Agent ◄─────┘
       │
       ▼
  个性化学习方案推送给学生
```

## 🛠️ 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Vue 3 + Vue Router + Chart.js + CSS3 |
| 后端 | Python FastAPI + Uvicorn |
| 数据库 | SQLite + aiosqlite |
| 认证 | JWT (python-jose) + bcrypt |
| AI/LLM | 可插拔设计（支持Mock/本地/云端LLM） |
| 部署 | Docker + Docker Compose |

## 🚀 快速开始

### 环境要求
- Python 3.9+
- pip

### 安装运行

```bash
# 1. 进入项目目录
cd ai-education-platform

# 2. 安装依赖
pip install -r backend/requirements.txt

# 3. 启动服务
python backend/main.py
```

访问 http://localhost:8000 即可使用。

### 或使用启动脚本

```bash
chmod +x start.sh
./start.sh
```

### Docker 部署

```bash
docker-compose up -d
```
